Neural Style Tensorflow:

Link: https://github.com/cysmith/neural-style-tf

Requirements:
- CUDA 8.0
- Cudnn 5.1
- Scipy
	> pip3 install --upgrade scipy
- OPENCV
	> pip3 install --upgrade opencv-python
- TensorFlow
	> CPU-only: pip3 install --upgrade tensorflow
	> GPU: pip3 install --upgrade tensorflow-gpu

Weights for Very Deep Convolutional Networks for Large-Scale Visual Recognition:
- Download: http://www.vlfeat.org/matconvnet/models/imagenet-vgg-verydeep-19.mat
- Save in project root directory

Simple example:
bash stylize_image.sh ./image_input/lion.jpg ./styles/kandinsky.jpg

Advanced example:
python3 neural_style.py --content_img lion.jpg --style_imgs kandinsky.jpg --max_size 1000 --max_iterations 100 --original_colors --device /cpu:0 --verbose
